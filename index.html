Best practices

  1. Don't use global variables/objects
  2. Declare variables at top of scope
  3. Use short clear meaningful names (English)
  4. Work in strict mode
  5. camelCase your code if(code != Constructor || CONSTANTS)
  6. Place external scripts at the bottom of the page
  7. Indent your code
